## Tecnologias

* **Java 11**
* **Spring Boot**
* **API REST**
* **PostgreSQL (Container e Heroku Postgres)**
* **RabbitMQ (Container e CloudAMQP)**
* **Docker**
* **docker-compose**
* **JWT**
* **Spring Cloud OpenFeign**
* **Heroku**
* **Coralogix Logging**
* **Kibana**

### Execução docker-compose

`docker-compose up --build`

Para ignorar os logs, adicione a flag `-d`.
